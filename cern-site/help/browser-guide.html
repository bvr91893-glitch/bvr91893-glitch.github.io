Help on the Browser You Are Using

Browsers for the early web ranged from simple line-mode tools to graphical interfaces. They all shared a basic pattern: choose a link, follow it, then continue navigating.

  • The line-mode browser was lightweight and portable.
  • Graphical browsers added windows, richer formatting, and easier navigation.
  • Browser software depended on the hardware and operating system available.
