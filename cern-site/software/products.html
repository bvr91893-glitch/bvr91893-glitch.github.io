Software Products

This page summarizes important software components connected with the early WorldWideWeb effort.

Line Mode Browser

A text-based browser designed to run on many systems and prove the portability of the web.

X11 Viola

A graphical browser approach for X Window environments that showed richer presentation possibilities.

NeXTStep Browser

The original browser-editor developed on the NeXT platform and central to the first demonstrations.

Servers

HTTP servers and gateways made project documents available and connected existing systems.

Tools

Supporting tools included converters, link utilities, and browser-related helpers.

Mail Robot

Mail-based services helped users retrieve information when direct interactive access was limited.

Library

The shared library code provided reusable networking and document-handling pieces for different browsers and servers.
